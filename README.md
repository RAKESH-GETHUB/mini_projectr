# Sentiment Analysis using NLP

## Project Overview

This project performs **Sentiment Analysis** on Tweets or Reddit posts related to trending topics such as:

* Cricket Matches
* Product Launches
* Movie Reviews
* Social Media Trends

The system classifies user opinions into:

* Positive
* Negative
* Neutral

The project also visualizes public opinion using charts and graphs.

---

# Features

* Text Cleaning and Preprocessing
* Sentiment Classification using NLP
* Positive/Negative/Neutral Detection
* Bar Chart Visualization
* Pie Chart Visualization
* Word Cloud Generation
* Public Opinion Trend Graph

# Installation

## Step 1: Clone Repository

```bash
git clone https://github.com/your-username/sentiment-analysis.git
```

## Step 2: Open Project Folder

```bash
cd sentiment-analysis
```

## Step 3: Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn nltk textblob wordcloud
```
